# Telegram Scraper and Discord Poster

This project is a C# application that scrapes data from Telegram and posts it on Discord using webhooks.

## Prerequisites

- .NET Core 3.1 or later
- A Telegram bot token
- A Discord webhook URL

## Libraries Used

- Telegram.Bot for interacting with the Telegram API[1][5]
- CSharpDiscordWebhook for sending messages to Discord via webhooks[2][6]

## Setup

1. Clone this repository to your local machine.
2. Open the project in your preferred C# IDE (like Visual Studio).
3. Install the necessary NuGet packages (`Telegram.Bot` and `CSharpDiscordWebhook`).

## Usage

1. Replace `"api"` and `"YOUR_DISCORD_WEBHOOK_URL"` with your actual Telegram bot token and Discord webhook URL.
2. Run the application.
